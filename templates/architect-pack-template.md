# Architect Pack Template

Use this template to provide project documentation and planning updates as a single importable file.

Do not include secrets, API keys, passwords, tokens, credentials, or private configuration.

Each file section must use this exact delimiter format:

```text
============================================================
FILE: planning/STATE.md
============================================================

[file content]
```

============================================================
FILE: docs/PROJECT_OVERVIEW.md
============================================================
